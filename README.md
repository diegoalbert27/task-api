# Task Api With NestJs

Development Task Api tutorial with NestJs

## Download or clone project

```bash
$ git clone 
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Task Functionalities

- Create a task
- Get all task created
- Get task by id
- Delete task by id
- Update task by id

Require insomnia for api collection in ``api-collection-task-api.json`` file.
